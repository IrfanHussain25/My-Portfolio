## 📌 Setup & Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/IrfanHussain25/My-Portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd My-Portfolio
   ```
3. Open `index.html` in your browser or use Live Server (VS Code extension)
